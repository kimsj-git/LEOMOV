# LEOMOV
- 목적: 다양한 컨텐츠의 1대1 대결을 통한 순위 제공 서비스
- 대표 기능: 영화 1대1 매치(투표) 승률에 따른 리그 순위 실시간 제공
- 확장 기능: 영화뿐만 아닌 다양한 컨텐츠로 리그를 생성하는 기능
- 그외 기능: 커뮤니티(매치 댓글), 좋아요, 프로필
- 달성 목표: 기획 -> 개발 -> 배포까지 해보기
